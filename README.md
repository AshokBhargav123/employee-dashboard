# Employee Management Dashboard

##  Project Overview
The Employee Management Dashboard is a React.js application that allows users to manage employees efficiently.  
It includes authentication, employee CRUD operations, image upload with preview, search, filtering, and print functionality, all presented in a clean dashboard-style UI.

---

##  Tech Stack
- React.js (JavaScript)
- React Router DOM
- Context API
- LocalStorage (for persistence)
- Plain CSS (No UI frameworks)

---

##  Features
- Mock login authentication with protected dashboard
- Add, Edit, and Delete employees
- Upload employee profile image with preview
- Search employees by name
- Filter employees by gender
- Filter employees by active / inactive status
- Combined filtering (search + gender + status together)
- Print employee list
- Responsive and clean dashboard UI
- Data persistence using LocalStorage
